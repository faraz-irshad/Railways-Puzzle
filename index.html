Game Rules

There are no specific requirements for placing railway elements, so you can decide how to approach this part. Here are some ideas:

  • Clicking on cells cycles through different element types, with options to rotate or delete using the right mouse button.
  • After selecting a tile, elements can be chosen from a palette or switched using the arrow keys.
  • Elements can be dragged and dropped onto a tile from a palette.
  • The railway line can be drawn by holding down the mouse button and moving across the cells.
